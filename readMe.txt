Program description: This program represents a social network of students as an undirected graph. It provides
    functionalities to manage students, friendships, and perform operations such as removing friendships,
    deleting accounts, displaying friend circles, and calculating closeness centrality.

Running the program:
    The program first asks the user for file name to populate graph. Once graph is successfully read, the user is
    presented with an option menu that allows the user to perform several tasks and find info regarding the graph/social
    network.
